Project Details

This project is a minimal offline-capable classroom demo. Data is stored locally; quizzes and lessons can be exported/imported as JSON or embedded in PNG files for convenient transfer.

Features

  • Multi-page structure: Home, Library, Open Classroom, Quizzes, Project Details
  • Local storage for lessons & quizzes
  • Export/import JSON and PNG-embedded data
  • Service Worker for offline usage
  • Mobile-friendly, touch-first controls

How the PNG export works

A small marker and base64 JSON are appended to a valid PNG file. Most image viewers ignore trailing bytes so the file remains an image while carrying your quiz data inside.
